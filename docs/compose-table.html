The Linux AltGr and Compose Key Masterclass (09.11.2007 edit)
This post is brought to you by Kubuntu 7.04, on the grounds that I don't have the live CD for 7.10 yet, which should be released round about now. It is also brought to you by the use of the Linux compose key, which I have previously worked out.

So far I know how I can write obscure Eastern European characters using the compose key, thus bringing you Skyforger track titles in Latvian such as Kad Ūsiņš Jāj and Tīreļa Purvā.

What I have yet to work out is how to produce Cyrillic characters. Normally in Windoze I would head straight for the Character Map and double-click on them one by one to produce such band names as Балканджи and Тролль Гнёт Ель. (I had to copy those directly from balkandji.org and www.trollge.ru).

I have yet to find a character map on Kubuntu; so far I've only seen anything remotely like the Windoze version on Mandriva, which is not my distribution of choice. So how's it done, short of changing the whole keyboard to a Russian layout?

Incidentally, I plan to use this post as a reference to what the compose key is capable of... and it will be updated as I find more tricks. I've also (09.11.2007) found that the AltGr key cuts a few of these even shorter...

EDIT: Kubuntu's Russian keyboard layout does work, but the positions of the characters are anything but similar to English. The Bulgarian layout is also very different to Russian, and though many Bulgarians are reported to use a phonetic keyboard somewhat resembling a standard QWERTY (e.g. ASDFG = АСДФГ), Kubuntu uses the standard Bulgarian layout. Furthermore, Ё and Ы are missing from the Bulgarian layout.
